# Currency Converter
A Python application that converts one currency to another using real-time exchange rates.

# Features
- Real-time exchange rates
- Supports multiple currencies
- Easy-to-use interface
- Accurate currency conversion

# Requirements
- Python 3.x
- requests library

# Installation
Install dependencies:
pip install requests

# How to Run

1. Download the project files.
2. Open a terminal in the project folder.
3. Run:
currency_converter.py
4. Enter:
   - Base Currency (USD, INR, EUR, etc.)
   - Target Currency
   - Amount

# Sample Output
